# HTML & CSS Quiz Game UI

This project is a static front-end interface for a **quiz game website** built using **HTML** and **CSS**. It was originally created as a university coursework project and includes multiple pages such as the home page, sign-in page, sign-up page, game page, scores page, help page, and logout page.

The project focuses on page layout, navigation, styling, and user interface design. It does not include full JavaScript game functionality or backend/database integration.

---

## Project Timeline

* **Originally completed:** 2024
* **Refactored and published on GitHub:** 2026
* **Context:** Web development coursework project

This repository has been cleaned and documented for portfolio purposes.

---

## Project Status

This is an early-stage static front-end project. It demonstrates HTML/CSS structure and UI design for a quiz game concept, but it is not a fully functional quiz application.

It is included in this portfolio to show my progression in front-end development, web page structure, styling, and interface design.

No installation is required.

Alternatively, if using VS Code, you can open the project with the Live Server extension.
